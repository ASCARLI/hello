#remove()函数修改报告
我修改了remove()函数。
##功能
**（加粗为修改设计部分）**
1. 若当前位置为空则不进行操作；
2. 若链表仅有一个元素则删除此元素成为空链表；
3. **若链表有超过一个元素且当前位置不为第一个，则删除当前元素，且currentpos更新为原来的前一个元素**
